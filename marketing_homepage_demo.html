Quick Setup

Get your Rails SaaS application up and running in minutes with our guided setup process.

Installation Steps

  1. Clone the repository: 
    git clone https://github.com/mitchellfyi/rails-starter.git
cd rails-starter
  2. Install dependencies: 
    bundle install
yarn install
  3. Setup the database: 
    rails db:create
rails db:migrate
rails db:seed
  4. Configure environment: 
    cp .env.example .env
# Edit .env with your configuration
  5. Add modules as needed: 
    bin/synth add ai
bin/synth add billing
bin/synth add cms
  6. Start the development server: 
    bin/dev
